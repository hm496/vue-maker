# Vue Maker

Create Vue apps with no build configuration.

## Quick Start

```sh
npm i -g vue-maker
mkdir my-app
cd my-app
vue-maker init
vue-maker start
```

## Build

```sh
vue-maker build
```

##  Directory structure
```
my-app
├── package.json
├── .gitignore
└── src
    └── App.vue
    └── index.html
    └── index.js
    └── index.less
```
